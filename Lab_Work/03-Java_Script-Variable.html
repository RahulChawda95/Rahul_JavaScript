Variable/Identifier:  it is used to store values

	# Three ways to declare a variable

	1. var (default and Global)
	2. let (works with scope)
	3. const (fix value)

	Rules:

	1. var name cant start with a number
	2. var name cant be keywords

	Syntex:
	document.write(variable-Name);
	console.log(variable-Name);
	document.getElementById('idname').innerHTML = variable-NAme;
Input: 
<script type="text/javascript">
	1)Variable

	var x=-89.34;== Declaration
	var x=85.56;== Re- Declaration
	x=55;== Re-assignment
	Note:All Allowed


	document.write(x)== Print inside body/page.
	document.write("<br>")== Taking Break line.

	2)let 

	let y=78;== Declaration
	// let y=-78.45;== Re- Declaration NOT ALLOWED
	y=45== Re-assignment ALLOWED

	document.write(y)== Print inside body/page.
	document.write("<br>")== Taking Break line.

	3)Constant

	const z=65;== Declaration
	// const z=82;== Re-Declaration NOT allowed
	// z=65;== Re-assignment NOT allowed

	document.write(z)== Print inside body/page.
	document.write("<br>")== Taking Break line.
	
</script>
