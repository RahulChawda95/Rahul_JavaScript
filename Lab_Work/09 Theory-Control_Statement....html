Theory Of Control-Statement 
		
Control-Statement

		
1)Conditional Statements

		
1)if Statement

		Note:This Is if Statement Using For Only One condition.
		syntax:
			if(Condition)
			{
				document.write(Something Text Here.....!)
			}
		
		
2) if else Statement

		Note:This Is if Statement Using For Two Condition.
		syntax:
			if(Condition)
			{
				document.write(Something Text Here.....!)
			}
			else
			{
				document.write(Something Text Here...!)
			}
		
		
3)if else if else  Statement

		Note:This Is if Statement Using For More Then Two Condition.
		syntax:
			if(Condition.)
			{
				document.write(Something Text Here...!)
			}
			else if(Condition.)
			{
				document.write(Something Text Here...!)
			}
			else
			{
				document.write(Something Text Here...!)
			}
		
		
4) nested if Statement

		Note:This Is if Statement Using For More Then Two Condition.
		syntax:
		if(Condition...)
		{
			if(Condition...)
			{
				document.write(Something Text Here...!)
			}
			else
			{
				if(Condition...)
				{
					document.write(Something Text Here...!)
				}	
			}
			.
			.
			.
			.
			.
		}
