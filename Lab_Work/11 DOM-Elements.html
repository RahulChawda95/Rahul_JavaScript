Finding HTML Elements

find the elements first. There are several ways to do this:
  1. Finding HTML elements by id
  2. Finding HTML elements by tag name
  3. Finding HTML elements by class name
  4. Finding HTML elements by CSS selectors
  5. Finding HTML elements by HTML object collections

<h3>1)Example Of Finding HTML Element By ID</h3>
	<p id="getID">Finding HTML Elements by Id</p>
	<p>This example of <b>getElementById</b> method.</p>
	<p id="addID"></p>
	<script type="text/javascript">

	input=document.getElementById('getID');---->Note:Element Use
		1st -->store  and get value Inside Input Variable
		

	document.getElementById('addID').innerHTML="This Add Some Text + "+input.innerHTML;---->for Output Value inside inner HTML

	</script>




<h3>2)Example Of Finding HTML Element By Tag Name</h3>

<p>Finding HTML Element By Tag name</p>

<p>This example of <b>getElementsByTag name</b> method.</p>

<p id="third"></p>
<script type="text/javascript">
	para=document.getElementsByTagName('p')--->Note:Element's Use
		1st -->store  and get value Inside para Variable
		

	document.getElementById('third').innerHTML="This 3rd Paragraph Adding Using JS.<br>"---->For Output Value Inside InnerHTML

	document.write(para[0].innerHTML+"+ This is Index Value Is 0<br>")
	
	document.write(para[1].innerHTML+"+ This is Index Value Is 1")
	
</script>


<h3>3)Example Of Finding HTML Element By Class Name.</h3>

</script>
